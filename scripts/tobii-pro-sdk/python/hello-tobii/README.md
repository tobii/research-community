## Description
This is a simple Hello World style script that gets gaze from the selected Eye Tracker and then prints it to the console.

## Prerequisites
For more info see: https://developer.tobiipro.com/python/python-getting-started.html

* Install Python 3.10.* (https://developer.tobiipro.com/tobiiprosdk/platform-and-language.html)
* Install Tobii Pro SDK python module ```pip install tobii_research```

## How to run
* ```python hello-tobii.py```
